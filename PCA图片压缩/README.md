# 任菜菜的小小小
### 基于PCA的图像压缩

#### 数据集：学校所构建数据集，不提供
#### 数据集数据集中有大约几百个人的图像，每个人人有大改十几张照片，选择其中2张昨晚基准，压缩后进行相似度匹配计算准确率
